Installation¶

Requirements¶

Required python packages would be auto installed while installing dms2dfe. Anaconda Python Distribution is recommended for installing dependent python packages.

Especially we recommend installing pysam prior installing dms2dfe.

conda install -c bioconda pysam==0.8.4

dms2dfe produces .pdb files with B-factor score equaling to Fitness values of corresponding residues. For making visualizations on PDB structures though dms2dfe pipeline, UCSF-Chimera (tested on 1.10.2) is needed to be installed. To use UCSF-Chimera through python environment, graphics drivers also need to be configured by installing “mesa-utils”.

Installation of dms2dfe¶

dms2dfe can be installed by following commands.

git clone https://github.com/kc-lab/dms2dfe.git
cd dms2dfe
pip install .

Troubleshoot¶

Please refer to Troubleshoot page. For information about dependencies (which are auto-installed), please refer to Dependencies.
